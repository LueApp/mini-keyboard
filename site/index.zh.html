为 4x3 宏键盘打造的 keyd 配置 + 实时 HUD

mini-keyboard

把通用 USB 迷你键盘 4132:2107 变成模态宏键盘： 应用快捷键、导航键，以及计算器风格的数字层。配套 HUD 会显示当前生效的 映射，空闲时折叠，并让你无需手写 keyd 配置即可编辑映射。只有这块键盘会被 重映射——其他所有键盘都保持原样，运行于 KDE Plasma 6 / Wayland。

开始使用

在终端里安装一次——之后从浏览器完成日常维护。

keyd 配置只需在终端里一次性安装。完成之后，如果你运行 web2local ——一个小型本地守护进程，只在你于原生对话框中批准后才执行白名单内的操作，且仅限回环地址—— 本页就会变成一个控制台，用于查看状态、校验、keyd 日志，以及安装或移除 HUD，无需终端。

手动安装

终端 · 一次性

安装脚本会把 config/default.conf 复制到 /etc/keyd/， 重启 keyd，并安装 HUD 编辑器的 root 辅助程序。以下四步只需执行一次。

  1. 克隆仓库 
    git clone https://github.com/LueApp/mini-keyboard.git
cd mini-keyboard
  2. 安装 keyd 
    sudo apt-get install -y keyd
  3. 应用迷你键盘配置 
    ./install.sh
  4. 启动 HUD 
    hud/install-hud.sh

web2local 控制台

检测中…

日常维护无需终端。每个按钮只运行 tools/web2local-minipad.sh 中一个 具名操作——绝不接受任意 shell 输入——并且 web2local 会先请你批准。

编辑准则

          
常规重映射请使用面板编辑器。

          

            点击 HUD 中的编辑控件，选择一个模式，然后点击任意按键格。保存流程会通过
            pkexec /usr/local/sbin/minipad-apply 写入，校验 keyd 文件，
            重启 keyd，并在守护进程报错时自动回滚。
          

        

        
    
          
  1. 
            进入编辑模式
            
    选择你想修改的模式标签，或新增一个带有专属双键组合键的模式。
    
          
    2. 
          
  2. 
            重映射按键
            
    选择一个预设，或输入原始 keyd 值，例如 C-c、up、5 或 macro(text enter)。
    
          
    3. 
          
  3. 
            安全保存
            
    辅助程序会拒绝符号链接输入、行内注释、不安全的模式名、重复的组合键，以及 keyd 的 command() 绑定。
    
          
    4. 
        

      

    


    

      

        

          
排错

          
映射异常时有用的检查手段。

        

        

          

            
配置错误

            
journalctl -eu keyd

          

          

            
查找设备 ID

            
sudo keyd.rvaiya monitor

          

          

            
观察模式切换

            
sg keyd -c 'keyd.rvaiya listen'

          

          

            
重映射后的输出

            
python3 tools/keyout.py

          

        

        

          紧急恢复：若按键卡死，按住 backspace + esc + enter 可终止 keyd 并恢复原始输入。
